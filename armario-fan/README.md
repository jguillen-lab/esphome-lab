# armario-fan

Controlador de ventiladores para armario basado en ESP32 y ESPHome.

## Qué hace

- Controla dos ventiladores PWM
- Usa temperatura importada desde Home Assistant
- Control PID independiente por ventilador
- Ajuste en caliente de parámetros PID
- Modo manual y automático
- Monitorización por web y Home Assistant

## Archivos

- `armario-fan.yaml` — configuración principal de ESPHome
